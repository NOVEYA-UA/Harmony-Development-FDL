### 1. Типы данных:

#### 1.1. Целочисленные типы:
- **Целое**: Определяет целочисленные значения без десятичных знаков.
- **Пример**: `целое число = 10`

#### 1.2. Вещественные типы:
- **Вещественное**: Определяет числа с плавающей запятой.
- **Пример**: `вещественное число = 3.14`

#### 1.3. Строковые типы:
- **Строка**: Определяет последовательность символов.
- **Пример**: `строка текст = "Привет, мир!"`

### 2. Операторы:

#### 2.1. Арифметические операторы:
- **Сложение**: Складывает два операнда.
- **Вычитание**: Вычитает один операнд из другого.
- **Умножение**: Умножает два операнда.
- **Деление**: Делит один операнд на другой.
- **Пример**: `результат = 10 + 5`

#### 2.2. Логические операторы:
- **И**: Возвращает true, если оба операнда истинны.
- **ИЛИ**: Возвращает true, если хотя бы один из операндов истинен.
- **НЕ**: Инвертирует значение операнда.
- **Пример**: `результат = (истина ИЛИ ложь)`

### 3. Управляющие структуры:

#### 3.1. Условные операторы:
- **Если-Иначе**: Выполняет блок кода, если условие истинно, иначе выполняет другой блок кода.
- **Пример**: 
```
если (условие) {
    выполнить это;
} иначе {
    выполнить другое;
}
```

#### 3.2. Циклы:
- **Цикл Пока**: Выполняет блок кода, пока условие истинно.
- **Цикл Для**: Выполняет блок кода определенное количество раз.
- **Пример**:
```
пока (условие) {
    выполнить это;
}
```

### 4. Функции и процедуры:

#### 4.1. Функции:
- **Функция**: Блок кода, который выполняет определенную задачу и возвращает значение.
- **Пример**: 
```
функция сложить(а, б) {
    вернуть а + б;
}
```

#### 4.2. Процедуры:
- **Процедура**: Блок кода, который выполняет определенную задачу без возврата значения.
- **Пример**:
```
процедура приветствие() {
    вывести("Привет, мир!");
}
```

### 5. Комментарии:

#### 5.1. Однострочные комментарии:
- **Однострочный комментарий**: Используется для добавления пояснений к коду в одной строке.
- **Пример**: `// Это комментарий`

#### 5.2. Многострочные комментарии:
- **Многострочный комментарий**: Используется для добавления пояснений к нескольким строкам кода.
- **Пример**: 
```
/*
Это
многострочный
комментарий
*/
```
